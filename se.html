Q5.What is software Designing?

Software designing is process which is used to design the blueprint of the software.

In blueprint, the requirements of the software are designed.

The blueprint gives us the complete details of the working software.

The design must be accurate on the user requirements.


Q6.What is software Reverse Engineering?

It is a process of analysing or understanding the basis method, design and requirements specifications of legacy software product.

In software reverse engineering first we analize the already create software , software source code and understand it after that recreating the software with some improvement.
