Q8.Explain Architecture of Multimedia.

There are three models is available in multimedia architecture:-
1.Standalone 2.Server 3.Networked

Both models have 4 layer Application, Media management, End to End Qos (quality of service) and last one is Storage subsystem

Application:- it is the first layer of multimedia architecture which is used to provide the interface between user and Computer system.

Media management:- it provides the synchronization of services which is intrect by user.

Synchronisation:- Synchronisation is a process of Sync data which are available in different different server.

End to End Oos:- it is the most important layer in multimedia architecture.

It's provide the QOS means Quality of service between source multimedia & destination multimedia content

Storage subsystem:- It's provide functionalities to multimedia services through centralise multimedia system...

Q10.Explain the Framework of Multimedia system.

A multimedia framework is a software framework that handles media on a computer through network.

It is used by an applications such as media players and audio or video editors.

But can also be used to build videoconferencing applications, media converters and other multimedia tools.

A multimedia framework provides a run time environment for the media processing.

Ideally such an environment provides execution contexts for the media processing blocks separated from the application using the framework.
